| `screenshot-daten.png` | Daten-Tab: AC-Leistung, Energie, Wirkungsgrad |

| `screenshot-system.png` | System-Tab (optional) |

## Ersetzen

1. Screenshots als PNG exportieren (Breite ca. 1200–1600 px, mobil reicht).
2. Dateien in diesen Ordner kopieren (`docs/screenshots/`) – **bestehende Dateien überschreiben**.
3. Commit & Push – das README verlinkt diese Pfade direkt.

Aktuell liegen teils noch **Platzhalter-Kopien** (v0.4.x Desktop) – mit deinen v0.6.4-Mobilbildern ersetzen.

**Tipp:** Kein Browser-Rahmen, dunkles Theme beibehalten – passt zum README-Layout.
